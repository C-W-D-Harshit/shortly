# Short.ly

A URL shortening service built with Next.js, TailwindCSS, and Shadcn/UI.

## env

Create a `.env` file in the `/server` directory with the following variables:

```bash
DATABASE_URL=
REDIS_URL=
REDIS_TOKEN=
PORT=3000
BASE_URL=http://localhost:3000
```

## To start this project

```bash
pnpm i
```

```bash
pnpm start
```
